Binder

interface Binder<JavascriptEngine, EncodedType>

Binder interface for the generated Plugin binder class. JavascriptEngine represents the class which is capable of executing javascript, such as an Android WebView or V8. EncodedType represents the encoded type the JavascriptEngine expects.
